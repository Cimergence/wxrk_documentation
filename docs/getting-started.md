# Getting Started with WXRK

**WXRK** is an AI-powered job application platform that helps candidates generate tailored, high-quality CVs and cover letters in minutes. Instead of rewriting your CV for every application, WXRK learns who you are—your experience, skills, and personality—then produces job-specific documents that land interviews.

## What you can do with WXRK

| Feature | Description |
|---|---|
| **Profile** | Store your personal details and MBTI personality type |
| **Technical Review** | Build a rich database of your work experience, projects, and STAR stories |
| **Applications** | Create job applications by pasting a job description or a URL |
| **CV Generation** | Get an AI-optimised CV matched to the job description |
| **Cover Letter** | Get a personalised cover letter written in your voice |
| **Dashboard** | Track all your applications and generation activity |

## The candidate flow at a glance

```
Sign up / Login
     │
     ▼
Onboarding (4 steps)
  ① Welcome
  ② Your Profile (name, location, LinkedIn)
  ③ Personality (MBTI test or override)
  ④ Ready
     │
     ▼
Technical Review
  • Add past jobs (company, role, dates)
  • Add projects with tech stack and impact metrics
  • Add STAR stories (Situation / Task / Action / Result)
  • Rate skill proficiency
     │
     ▼
New Application
  • Paste or URL-fetch a job description
  • Add a personal note (optional)
     │
     ▼
Generate
  • CV — optimised against the job description
  • Cover Letter — written in your MBTI tone
  • Both at once
     │
     ▼
Review & Export
  • Edit the generated text
  • Export as PDF
  • Track application status (Interview → Offer / Rejected)
```

## Prerequisites

- A modern browser (Chrome, Firefox, Safari, Edge)
- An account on WXRK (email or OAuth login)

## Quick start

1. Go to the WXRK app and create an account or log in.
2. Complete the **[Onboarding](./guide/onboarding)** wizard — it takes about 3 minutes.
3. Build your **[Technical Review](./guide/technical-review)** — the richer your data, the better the output.
4. Create your first **[Application](./guide/applications)** and generate your documents.
